# Calc-IT_management

[![CI/CD Pipeline](https://github.com/killziak/Calc-IT_management/actions/workflows/main.yml/badge.svg)](https://github.com/killziak/Calc-IT_management/actions/workflows/main.yml)

Вебзастосунок-калькулятор, для якого налаштовано CI/CD pipeline на GitHub Actions та автоматичний деплой через Vercel.

## Production URL
https://vercel.com/killziaks-projects/calc-it-management

## CI/CD
- GitHub Actions: lint + unit tests + build
- Vercel: automatic deployment from main

## Features
- арифметичні операції
- відсотки
- зміна знаку
- підтримка dev/prod mode через environment variables
